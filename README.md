# faers-pharmacovigilance-analysis
Exploratory pharmacovigilance analysis using FAERS data (2025 Q1) focusing on adverse events, demographics, outcomes, and signal detection (PRR).
## Project Highlights

- Cleaned and processed raw FAERS ASCII datasets (DEMO, DRUG, REAC)
- Built relational joins using primary keys (primaryid)
- Performed exploratory data analysis (EDA) on demographics and adverse events
- Implemented signal detection using:
  - Proportional Reporting Ratio (PRR)
  - Chi-square test
- Conducted subgroup analysis (age, sex)
- Visualized rate ratios and demographic distributions using matplotlib
- Applied statistical testing (t-test, chi-square) for comparative analysis

## Technical Skills Demonstrated

- Python (pandas, numpy, matplotlib, scipy)
- SQL (joins, aggregations, filtering)
- Data cleaning & preprocessing
- Statistical hypothesis testing
- Exploratory Data Analysis (EDA)
- Pharmacovigilance signal detection logic

## Key Finding Example

The Vedolizumab–Diarrhoea drug–event pair showed elevated disproportionality (PRR > 1) with statistically significant chi-square results, suggesting a hypothesis-generating safety signal.
